### Опис проєкту

Тема проєкту: Магазин компонентів комп’ютера  
Мета: створити застосунок для перегляду, пошуку та покупки комп’ютерних комплектуючих.  
Мова програмування: Python, C++, HTML
Інструменти: Git, GitHub, Flask або Django  

Користувач може:
- переглядати каталог товарів;  
- додавати товари до кошика;  
- переглядати суму замовлення.  

Адміністратор може:
- додавати, редагувати й видаляти товари.

---

## Розроблені модулі

| Модуль | Призначення |
|--------|--------------|
| **main.py** | Головний файл застосунку, викликає інші модулі |
| **catalog.py** | Каталог товарів, перегляд і додавання до кошика |
| **cart.py** | Робота з кошиком (додавання, видалення, оформлення) |
| **admin.py** | Панель адміністратора для керування товарами |
| **database.py** | Збереження даних про товари (список або JSON) |
| **user.py** | Модуль користувача (реєстрація, історія покупок) |

---

## Функціональність модулів

- **main.py** — головне меню застосунку  
- **catalog.py** — вивід списку товарів  
- **cart.py** — управління кошиком  
- **admin.py** — редагування асортименту  
- **database.py** — робота з базою даних  
- **user.py** — обробка користувачів
