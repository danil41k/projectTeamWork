# Магазин комп'ютерних комплектуючих (Flask)

## Опис
Простий навчальний Flask-застосунок з 4 сторінками:
- Головна
- Про нас
- Товари
- Контакти

## Як запускати
1. Встановити залежності:
   ```
   pip install flask
   ```
2. Запустити:
   ```
   python app.py
   ```
3. Відкрити в браузері: http://127.0.0.1:5000/
